# omz-mytheme

zsh에서 사용하는 개인 테마입니다.
테마 파일의 경로는 ~/.oh-my-zsh/themes/mytheme.zsh-theme 입니다

plugin은 https://github.com/olets/zsh-transient-prompt 을 다운로드 받아서 ~/.oh-my-zsh/custom/plugins/transient-prompt 경로에 설치합니다.
해당 폴더에 transient-prompt.plugin.zsh 파일이 있어야 합니다.
.zshrc 파일에 plugins에 transient-prompt를 추가해서 사용하지만 현재 스크립트 버전에서는 정상동작하지 않습니다.
